# 🌸 Daily Habit Tracker

แอปบันทึกกิจกรรมประจำวัน + ตั้ง "กิจกรรมวนซ้ำทุก N วัน" ได้อย่างอิสระ
(เช่น สระผมทุก 2 วัน โดยวันจะขยับไปเรื่อยๆ ไม่ตายตัวว่าเป็นวันไหนของสัปดาห์
— จันทร์ → พฤหัส → อาทิตย์ → พุธ → เสาร์ → อังคาร → ศุกร์ → จันทร์ วนไปเรื่อยๆ)

ต่างจากแอปทั่วไปที่มักให้ตั้งได้แค่ "ทุกวันจันทร์" หรือ "ทุก 7 วัน"
แอปนี้ให้ตั้ง **ทุก N วันจริงๆ** นับต่อเนื่องจากวันเริ่มต้น เหมาะกับกิจวัตรที่ไม่ได้อิงวันในสัปดาห์

## ฟีเจอร์

- ➕ เพิ่มกิจกรรมวนซ้ำได้ไม่จำกัดจำนวน กำหนดเองว่าทำทุกกี่วัน
- ✅ ดูว่าวันนี้ต้องทำอะไรบ้าง กดติ๊กว่าทำแล้ว
- 📝 เขียนบันทึก/ไดอารี่ประจำวันว่าวันนี้ทำอะไรไปบ้าง
- 📖 ดูประวัติย้อนหลังทั้งหมด
- 🎨 หน้าตาน่ารัก ใช้งานง่าย (ทำด้วย Streamlit)

## วิธีติดตั้งและรัน

ต้องมี Python 3.9 ขึ้นไป

```bash
# 1. โคลนโปรเจค
git clone https://github.com/<your-username>/habit-tracker.git
cd habit-tracker

# 2. ติดตั้ง dependencies
pip install -r requirements.txt

# 3. รันแอป
streamlit run app.py
```

จากนั้นเบราว์เซอร์จะเปิดขึ้นมาอัตโนมัติที่ `http://localhost:8501`

## โครงสร้างโปรเจค

```
habit-tracker/
├── app.py               # หน้าตาแอป (Streamlit UI)
├── db.py                # จัดการฐานข้อมูล SQLite
├── schedule_utils.py    # คำนวณวันที่ถึงกำหนดของกิจกรรมวนซ้ำ
├── requirements.txt
├── .gitignore
└── README.md
```

ข้อมูลทั้งหมดจะถูกเก็บไว้ในไฟล์ `habit_tracker.db` (SQLite) ที่สร้างขึ้นอัตโนมัติ
เมื่อรันแอปครั้งแรก (ไฟล์นี้ไม่ถูกอัพขึ้น GitHub เพราะอยู่ใน `.gitignore`)

## หลักการคำนวณวันที่ถึงกำหนด

กิจกรรมแต่ละอย่างมี "วันเริ่มนับ" (start date) และ "ทำทุกกี่วัน" (interval)
ระบบจะเช็คว่า `(วันนี้ - วันเริ่มนับ) % interval == 0` ถ้าใช่แปลว่าวันนี้ถึงกำหนด
วิธีนี้ทำให้วันที่ถึงกำหนดขยับไปเรื่อยๆ ตามจำนวนวันจริง ไม่ผูกกับวันในสัปดาห์
